# Roshani Chaurasiya | Portfolio

A professional, dynamic, and elegantly designed portfolio website to showcase my skills, projects, and experience as a Python Developer and MERN stack enthusiast.

## Features
- **Modern UI/UX**: Premium dark theme with vibrant crystal-red accents (`#FF0000`).
- **3D Interactive Backgrounds**: Powered by `three.js` for an immersive visual experience.
- **Custom Cursor**: Interactive red cursor that responds to elements.
- **Responsive Layout**: Designed to look great on desktop and mobile devices.
- **Project Showcase**: Clean grid system highlighting top projects with live and source code links.

## Technologies Used
- HTML5
- Vanilla CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (ES6, Web Audio API, Canvas API)
- Three.js (for 3D graphics)
- Typed.js (for text animation)
- EmailJS (for contact form functionality)

## How to Run Locally

You can simply open `index.html` in your web browser. Or, to avoid CORS issues and view the 3D graphics perfectly:

1. Clone this repository
2. Use an HTTP server like `serve`:
   ```bash
   npx serve .
   ```
3. Open the provided `localhost` link in your browser!
